  GameSpeed Trainer for Dying Light 1
  by shotilive
  ─────────────────────────────────────────────────────────────

WHAT THIS IS
─────────────────────────────────────────────────────────────
A real-time game speed controller for Dying Light 1.
Slow the world down for cinematic moments, speed it up for
chaos, or hit 100x and watch everything go absolutely insane.

No injection. No cheat engine. No bans.

─────────────────────────────────────────────────────────────
REQUIREMENTS
─────────────────────────────────────────────────────────────
  - Dying Light 1 on Steam
  - Windows 10 or 11
  - That's literally it.

─────────────────────────────────────────────────────────────
INSTALLATION  (do this once)
─────────────────────────────────────────────────────────────
  1. Extract this zip to anywhere on your PC
     (Desktop, Downloads, wherever - keep the folder intact)

  2. Right-click Install.exe
     -> Run as administrator

  3. The installer finds Dying Light automatically.
     If it doesn't, click Browse and point it to the folder
     that contains DyingLightGame.exe

  4. Click INSTALL

  5. Done. Close the installer.

─────────────────────────────────────────────────────────────
HOW TO USE
─────────────────────────────────────────────────────────────
  1. Launch Dying Light via Steam as normal
  2. Load into gameplay
  3. Run "SHOTI GameSpeed Trainer.exe" from this folder
  4. Status indicator turns GREEN when connected
  5. Use the slider or buttons to adjust speed

  PRESET BUTTONS:
    0.25x  -  slow motion
    0.50x  -  half speed
    1.00x  -  normal (default)
    4.00x  -  fast forward

  100x SPEED BUTTON:
    Click once  -> activates 100x (button stays lit)
    Click again -> turns off, returns to previous speed

  HOTKEYS (work while in-game, trainer just needs to be open):
    F1  ->  Normal    (1.00x)
    F2  ->  Lowest    (0.05x)
    F3  ->  Slow-mo   (0.25x)
    F4  ->  Fast      (4.00x)

  The trainer resets speed to 1.0x automatically when closed.

─────────────────────────────────────────────────────────────
PLAYING ONLINE / CO-OP
─────────────────────────────────────────────────────────────
  Disable VAC in Dying Light before playing online:
    Options -> Online -> VAC Secure Game: No

  Speed changes only affect the host. Guests see normal speed.

─────────────────────────────────────────────────────────────
UNINSTALLING
─────────────────────────────────────────────────────────────
  Go to your Dying Light game folder
  (the folder containing DyingLightGame.exe) and delete:

    winmm.dll
    winmm_real.dll

  Done. Dying Light returns to 100% stock behaviour.
  You can also delete the trainer folder whenever you want.

─────────────────────────────────────────────────────────────
SOMETHING BROKEN?
─────────────────────────────────────────────────────────────
  Trainer shows red / not connecting
  -> Make sure Dying Light is fully loaded into gameplay,
     not just sitting on the main menu

  Game crashes on launch after installing
  -> You may be on an unsupported game version
  -> Delete winmm.dll from the game folder to restore it

  Speed feels off / wrong
  -> Hit F1 or click 1.00x to reset to normal
